CribSmack

A companion app to offline cribbage games which allows users to login to their profile and input game scores of two players, which render in-game averages and statistics. End-game data is rendered to each players' profile and aggregated with existing averages. Leaderboards track all users and rankings for each statistic. MERN RESTful app utilizing AJAX and relational database.

The LittleFreeLibarary.org Little Helper

The Free Little Library Little Helper app was built for Littlefreelibrary.org to assist their mission in building local communities through literacy. Utilizing technologies MySQL, Express.js, Node.js, and the Google Maps API, this website allows users to create a profile which displays books currently held in their Littlefreelibrary.org local library. Users will also input books deposited and track those checked out, as well as leave reviews and ratings for each book specific to their neighborhood library, creating a "neighborhood book club."
